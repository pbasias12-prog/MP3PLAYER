JavaFX MP3 Player Application
Developed a desktop MP3 Player application using JavaFX, IntelliJ IDEA, and Scene Builder
Designed an interactive graphical user interface with media controls, playlist management, and song navigation
Implemented audio playback functionalities including play, pause, next/previous track, volume control, and playback speed adjustment
Integrated JavaFX MediaPlayer API for multimedia handling and dynamic song updates
Focused on responsive UI design and user-friendly music playback experience
