# JavaFX MP3 Player

Desktop MP3 Player application developed using JavaFX, Scene Builder, and IntelliJ IDEA.

## Features
- Play and pause music
- Next and previous track navigation
- Volume control
- Playback speed adjustment
- Playlist management
- Dynamic song title updates
- Interactive graphical user interface

## Technologies Used
- Java
- JavaFX
- JavaFX MediaPlayer API
- Scene Builder
- IntelliJ IDEA
